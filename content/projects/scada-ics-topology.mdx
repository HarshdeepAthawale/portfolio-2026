A topology-discovery system for **OT / industrial control networks** that maps the network without introducing new risk.

## The problem

Most network-discovery tooling leans on protocols like **CDP/LLDP** — convenient, but insecure and unsuitable for sensitive ICS/SCADA environments. In operational technology, the discovery mechanism itself can become an attack surface.

## The approach

- Built a **non-intrusive** discovery pipeline aligned with the **Purdue Model (Levels 0–5)** and **NIST** compliance standards.
- **Eliminated CDP/LLDP** in favour of **authenticated telemetry**: SNMPv3, NetFlow, and Syslog for real-time OT visibility.
- Enforced **zero-trust access control** with IAM.

## Stack

AWS (Lambda, RDS, MQTT) · PostgreSQL · Grafana Node Graph · IAM
